# 远航用户系统
> 本用户系统为nodejs开发

## 部署说明

> .env 文件为环境变量文件，运行时务必修改环境变量配置文件

## 特性支持

1. 多应用

## 路由说明

1. [/api/user](/controllers/userController.js) 用户路由
2. [/api/app](/controllers/appControllers.js) 应用路由
3. [/api/admin](/controllers/adminController.js) 管理员路由
