# 远航用户系统
> 本用户系统为nodejs开发

## 部署说明

> .env 文件为环境变量文件，运行时务必修改环境变量配置文件

## 特性支持

1. 多应用

## 路由说明

3. [/api/admin](/controllers/adminController.js) 管理员路由
